# 亚细亚零售 AI 智能系统 POC

亚细亚零售企业 AI 智能化转型概念验证项目，包含**经营数据分析系统**和**内容营销系统**两大核心模块。

## 项目架构

```
POC/
├── 经营数据系统/          # 经营数据分析与智能决策平台
├── 内容营销系统/          # AI 驱动的内容营销自动化平台
└── README.md
```

## 经营数据系统

面向零售企业的经营数据分析与智能决策平台，整合 POS 收银、ERP 采购、CRM 会员等多源数据，提供实时经营监控、异常预警和 AI 归因推理。

### 核心功能

| 模块 | 功能说明 |
|------|---------|
| 经营总览 | 日/周/月多维度经营指标展示，AI 智能经营分析，品类营收分布，趋势图表 |
| 数据管理 | 多数据源接入管理（OA、企业微信、POS、ERP、CRM、SCM、HR） |
| 异常监控 | 可配置异常规则引擎，实时预警（营业额下降、毛利率异常、客流量波动等） |
| Agent 归因推理 | 跨系统自动归因分析，生成原因诊断与解决方案，自动推送通知和工单 |
| Agent 智能对话 | 自然语言交互式经营数据查询与分析 |

### 技术栈

- **前端**: React 18 + Ant Design 5 + Recharts + Vite
- **后端**: Express.js + better-sqlite3
- **数据**: SQLite（内置 44 天模拟经营数据）

### 快速启动

```bash
cd 经营数据系统

# 安装依赖
npm install
cd server && npm install && cd ..

# 启动（方式一：一键脚本）
start.bat

# 启动（方式二：手动启动）
# 终端1 - 后端
cd server && node index.js

# 终端2 - 前端
npx vite --host
```

启动后访问 http://localhost:3002

## 内容营销系统

AI 驱动的内容营销自动化平台，支持智能内容创作、营销日历管理、知识库运营和数据分析。

### 核心功能

| 模块 | 功能说明 |
|------|---------|
| 仪表盘 | 营销活动概览、内容产出统计、渠道效果分析 |
| 创作工作台 | AI 文案生成、AI 图片创作、AI 视频脚本生成 |
| 规则引擎 | 营销日历管理、定时任务调度 |
| 知识库 | 品牌素材库、营销话术库、产品知识库 |
| 数据分析 | 内容效果追踪、渠道 ROI 分析、用户画像洞察 |

### 技术栈

- **前端**: React 18 + Ant Design 5 + Recharts + Vite
- **后端**: Express.js + better-sqlite3
- **AI 服务**: 可接入 OpenAI / 本地大模型

### 快速启动

```bash
cd 内容营销系统

# 安装依赖
npm install
cd server && npm install && cd ..

# 启动（方式一：一键脚本）
start.bat

# 启动（方式二：手动启动）
# 终端1 - 后端
cd server && node index.js

# 终端2 - 前端
npx vite --host
```
